# Ollama and Openwebui Udemy Course
- [Course link](https://www.udemy.com/course/meta-llama-3-demystified-beginner-to-pro-with-ollama)
- [Link to OpenWebUI](http://localhost:8080)
- [Link to download and install Ollama](https://ollama.com/download)

**Start OpenWebUI**<br>
Run `$ open-webui serve`.

**Start a conversation**<br>
Run `$ ollama run llama3.2` or another [model](https://ollama.com/search)

**End a conversation**<br>
Run `$ /bye`.

Done.
